-DynamicDifficulty
-=================
-
-A Bukkit plugin that implements dynamic difficulty on a Minecraft 
-Bukkit server.
-
-If you plan on downloading the code and messing with it, this project 
-requires the following libraries in order to compile:
-   cglib-nodep-2.2.3.jar
-   easymock-3.1.jar
-   objenesis-tck-1.3.jar
-   craftbukkit-1.7.5-R0.1.jar
